Switcherblur

Configurable iOS 7 switcher snapshots blur.

Download .zip Download .tar.gz View on GitHub

Using a simple, low-level gaussian blur, SwitcherBlur allows for configurable switcher snapshot ("previews") blurring. By default, SwitcherBlur will blur every app preview in the iOS 7 switcher, however, by adding apps to the Settings' Blacklist area, it's possible to prevent specific apps from being affected. In addition, enabling "Inactive Blur" will only blur apps that aren't currently running in the background (unless Blacklisted), making the switcher ultimately far more useful and attractive.
